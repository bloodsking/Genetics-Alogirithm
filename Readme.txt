Cara memakai program kami

via google colab:
	1. download file .ipynb yang sudah kami sediakan di file
	2. buka "https://colab.research.google.com/?hl=id" dan upload file .ipynb yang telah di download
	3. setelah terbuka, tekan connect atau sambungkan pada bagian kanan atas 
	4. setelah muncul tulisan ram dan disk, kita bisa play satu per-satu
	5. jika ingin run secara keseluruhan bisa ke bagian runtime --> run all
	6. selesai program bisa dijalankan
